Build the CarouselBrain frontend MVP with an extremely polished, fluid, modern UI focused on the feeling of a premium AI-powered knowledge product.

The design language should feel like a blend of:

* Notion
* Readwise
* Linear
* Arc Browser
* modern Apple-esque smoothness

The product should feel:

* calm
* intelligent
* premium
* fluid
* minimal
* highly polished
* slightly futuristic but not flashy

DO NOT make it:

* neon cyberpunk
* generic AI SaaS template
* overloaded with gradients
* cluttered
* corporate

The UI should feel:
“Turning social media into a second brain.”

---

## PROJECT

Product name:
CarouselBrain

Core concept:
Users upload educational Instagram carousel screenshots.
The AI converts them into:

* summaries
* action steps
* concepts
* learning paths
* structured knowledge pages

The focus is:
knowledge extraction + organization.

---

## IMPORTANT

This phase is FRONTEND/UI ONLY.

DO NOT:

* integrate Supabase
* integrate OCR
* integrate Groq
* integrate real backend logic
* add authentication logic
* implement API routes

Use mocked/static data only.

The goal is:
validate UX, interactions, visual identity, and product feel.

---

## TECH STACK

Use:

* Next.js App Router

* Lucide React for icons

Do NOT use:

* shadcn

* Chakra
* Bootstrap

## DESIGN PRINCIPLES

The UI must feel:

* extremely smooth
* fluid
* responsive
* premium
* tactile
* fast

Animations should be:

* subtle
* spring-based
